# Project 1 - Registration Form Validation 📝

This is a simple Registration Form project created using **HTML, CSS, and JavaScript**.

The form contains **Name, Email, and Password** fields with real-time validation.  
The submit button will remain disabled until all inputs are valid.

---

## 🚀 Features

- Name field validation (cannot be empty)
- Email format validation (valid email required)
- Password validation (minimum 6 characters)
- Real-time error messages while typing
- Submit button disabled until all validations are correct
- Shows success alert after submitting

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM & Form Validation)
